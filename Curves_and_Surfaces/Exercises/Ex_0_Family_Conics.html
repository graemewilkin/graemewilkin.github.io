A family of conic sections

Back to Curves and Surfaces homepage

The picture below shows the curve \(x^2 + t y^2 = 1\), where \(t \in \mathbb{R}\) is a parameter. You can see how the curve changes with different values of \(t\) by clicking and dragging on the slider. In particular, if \(t > 0\) then the curve is an ellipse, if \(t < 0\) then the curve is a hyperbola, and if \(t = 0\) then the curve is a pair of straight lines. The applet will cycle through all the values of \(t\) if you play the animation (the button in the bottom left of the picture).
