Visualising the hyperboloid of one sheet using planar cross-sections

Back to Curves and Surfaces homepage

The following pictures show the intersection of the hyperboloid of one sheet \[ \{(x, y, z) \in \mathbb{R}^3 \, : \, x^2 +y^2 - z^2 = 1 \} \] with various planes. The first picture shows the intersection with the plane \(z = c\). You can move the slider to change the value of \(c\), and you can see how the curve of intersection (which is always a circle) changes.


The next picture shows the intersection with the plane \(ax + by = 0\) for different values of \(a\) and \(b\). As you can see, this intersection is always a hyperbola. This time you can move the slider to change the value of \(\theta\), which corresponds to \(\theta = -\arctan(a/b)\). Equivalently, the equation of the plane is \(\sin(\theta) x - \cos(\theta) y = 0\).
