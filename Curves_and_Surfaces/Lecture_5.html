Curvature

Let \(p : I \rightarrow \mathbb{R}\) be a smooth regular curve parametrised by arclength. Since the curve is parametrised by arclength, then \(|p'(s)| = 1\) for all \(s \in I\).

The unit tangent vector is defined to be \(T(s) = p'(s)\).

Since the tangent vector has unit length, then \(|T(s)|^2 = T(s) \cdot T(s) = 1\), and so \[0 = \frac{d}{ds} (T(s) \cdot T(s)) = T'(s) \cdot T(s) + T(s) \cdot T'(s) = 2T'(s) \cdot T(s) .\] Therefore \(T'(s)\) is orthogonal to \(T(s)\).

A point \(p(s)\) is a point of inflection if \(T'(s) = 0\).

If \(p(s)\) is not a point of inflection, then the unit normal vector is defined to be \(N(s) = \frac{T'(s)}{|T'(s)|}\). Note that the unit normal vector is well-defined if and only if \(p(s)\) is not a point of inflection.

Since \(|T(s)| = 1\) is constant along the curve, then the derivative \(T'(s)\) measures how much the curve is turning, and in which direction.

The curvature vector is defined to be \(k(s) = T'(s)\).
The curvature is the magnitude of the curvature vector, defined by \(\kappa(s) = |k(s)|\).

Intuitively, the curvature measures the rate at which the tangent vector changes direction. To gain a better understanding of curvature, it is useful to compute some examples.

Example. (The curvature of a circle)
Recall that \(p : \mathbb{R} \rightarrow \mathbb{R}^2\) given by \(p(s) = \left( r \cos \left(\frac{s}{r} \right), r \sin \left( \frac{s}{r} \right) \right)\) represents a circle of radius \(r\) parametrised by arclength. The unit tangent vector is \(T(s) = p'(s) = \left( -\sin \left( \frac{s}{r} \right), \cos \left( \frac{s}{r} \right) \right)\), and therefore the curvature vector is \[ k(s) = T'(s) = \left( -\frac{1}{r} \cos \left( \frac{s}{r} \right), -\frac{1}{r} \sin \left( \frac{s}{r} \right) \right) \] Therefore the curvature is \( \kappa(s) = |k(s)| = \frac{1}{r} \), and the unit normal vector is \(N(s) = \left( -\cos \left( \frac{s}{r} \right), -\sin \left( \frac{s}{r} \right) \right) \).

The above example tells us that the curvature of a circle is inversely proportional to the radius. We will explore this idea further for arbitrary curves when we study the topic of the osculating circle in the next lecture.

The next example is that of the helix, which also has constant curvature.

Example. (The curvature of a helix)
Recall that \(p : \mathbb{R} \rightarrow \mathbb{R}^3\) given by \(p(s) = \left( r \cos \left( \frac{s}{\sqrt{r^2+c^2}} \right), r \sin \left( \frac{s}{\sqrt{r^2+c^2}} \right), \frac{cs}{\sqrt{r^2+c^2}} \right)\) represents a helix parametrised by arclength. The unit tangent vector is \[ T(s) = p'(s) = \left( - \frac{r}{\sqrt{r^2+c^2}} \sin \left( \frac{s}{\sqrt{r^2+c^2}} \right), \frac{r}{\sqrt{r^2+c^2}} \cos \left( \frac{s}{\sqrt{r^2+c^2}} \right), \frac{c}{\sqrt{r^2+c^2}} \right) \] and we can differentiate again to obtain the curvature vector \[ k(s) = T'(s) = \left( - \frac{r}{r^2+c^2} \cos \left( \frac{s}{\sqrt{r^2+c^2}} \right), -\frac{r}{r^2+c^2} \sin \left( \frac{s}{\sqrt{r^2+c^2}} \right), 0 \right) \] Therefore the curvature is \( \kappa(s) = \frac{r}{\sqrt{r^2+c^2}} \) and the unit normal vector is \(N(s) = \left( -\cos \left( \frac{s}{\sqrt{r^2+c^2}} \right), -\sin \left( \frac{s}{\sqrt{r^2+c^2}} \right) , 0 \right)\).

Torsion

If \(\kappa(s) \neq 0\) so that \(p(s)\) is not a point of inflection, then the unit tangent vector and unit normal vector are orthogonal, and therefore their cross product is also a unit vector. Note that if \(p(s)\) is a point of inflection then \(N(s)\) is not well-defined.

The binormal vector is the unit vector \(B(s) = T(s) \times N(s)\).

The binormal vector is a unit vector orthogonal to both \(T(s)\) and \(N(s)\), and therefore the three vectors \(T, N, B\) form an orthonormal basis for \(\mathbb{R}^3\), called the Frenet frame of the curve at the point \(p(s)\).

The following picture shows the Frenet frame of the helix.
Since the binormal vector has constant length, then the following calculation shows that \(B'(s) \cdot B(s) = 0\). \[0 = \frac{d}{ds} (B(s) \cdot B(s)) = B'(s) \cdot B(s) + B(s) \cdot B'(s) = 2B'(s) \cdot B(s) \] Therefore \(B'(s)\) is perpendicular to \(B(s)\). (Note the similarity between this calculation and the analogous result for the tangent vector.)

We can also differentiate the definition of the binormal vector. Note that \(T'(s) = \kappa(s) N(s)\), and so \(T'(s) \times N(s) = \kappa(s) N(s) \times N(s) = 0\). \[ B'(s) = T'(s) \times N(s) + T(s) \times N'(s) = T(s) \times N'(s), \] and so \(B'(s)\) is orthogonal to \(T(s)\). Therefore, since \(T(s), N(s), B(s)\) form an orthonormal basis, and \(B'(s)\) is orthogonal to two of the basis vectors \(T(s)\) and \(B(s)\), then it must be parallel to the third vector \(N(s)\).

The torsion of the curve is the function \(\tau(s)\) such that \(B'(s) = \tau(s) N(s)\).

Example. (The torsion of the helix)
Continuing from the previous example, we can compute the binormal vector and hence the torsion for the helix. Recall that \[ T(s) = \left( - \frac{r}{\sqrt{r^2+c^2}} \sin \left( \frac{s}{\sqrt{r^2+c^2}} \right), \frac{r}{\sqrt{r^2+c^2}} \cos \left( \frac{s}{\sqrt{r^2+c^2}} \right), \frac{c}{\sqrt{r^2+c^2}} \right), \quad N(s) = \left( -\cos \left( \frac{s}{\sqrt{r^2+c^2}} \right), -\sin \left( \frac{s}{\sqrt{r^2+c^2}} \right) , 0 \right) \] Taking the cross product gives us the binormal vector \[ B(s) = T(s) \times N(s) = \left( \frac{c}{\sqrt{r^2 + c^2}} \sin \left( \frac{s}{\sqrt{r^2+c^2}} \right), -\frac{c}{\sqrt{r^2 + c^2}} \cos \left( \frac{s}{\sqrt{r^2+c^2}} \right), \frac{r}{\sqrt{r^2+c^2}} \right) \] Now we can differentiate this to compute the torsion \[ B'(s) = \left( \frac{c}{r^2+c^2} \cos \left( \frac{s}{\sqrt{r^2+c^2}} \right), \frac{c}{r^2+c^2} \sin \left( \frac{s}{r^2+c^2} \right) , 0 \right) = \frac{c}{r^2+c^2} N(s) . \] Therefore \(\tau(s) = \frac{c}{r^2 + c^2}\).
